#######################
|RDS| from AWS Explorer
#######################

|RDSlong| (|RDS|) is a service that enables you to provision and manage SQL relational database
systems in the cloud. |RDS| supports three types of database systems:

* MySQL Community Edition

* Oracle Database Enterprise Edition

* Microsoft SQL Server (Express, Standard, or Web Editions)

For more information, see the |RDS-ug|_.

A lot of the functionality discussed here is also available through the 
:console:`AWS Management Console <rds>` for |RDS|.
